# План интеграции авторизации через WhatsApp

## Обзор
Авторизация пользователей через WhatsApp работает по принципу, схожему с Telegram, но требует использования **WhatsApp Business API**. В отличие от Telegram Bot API, это не открытый и не полностью бесплатный протокол. Для работы требуются верифицированный бизнес-аккаунт и модель оплаты за диалоги (обычно первые 1000 диалогов в месяц бесплатно).

## Предварительные требования (Что нужно сделать вам)
1.  **Аккаунт Meta for Developers**: Создать приложение на сайте [developers.facebook.com](https://developers.facebook.com/).
2.  **WhatsApp Business Account (WABA)**: Настроить внутри панели управления Meta.
3.  **Номер телефона**: Нужен отдельный номер, который **не используется** в данный момент в личном WhatsApp или приложении WhatsApp Business.
4.  **Способ оплаты**: Привязать банковскую карту к аккаунту Meta (это обязательно, даже если вы планируете оставаться в рамках бесплатного лимита).

## План технической реализации

### 1. Конфигурация
*   Нам нужно добавить новые ключи в файл `application.conf`:
    *   `whatsapp.accessToken` (Рекомендуется использовать "Permanent System User Token" — постоянный токен системного пользователя).
    *   `whatsapp.phoneNumberId` (ID номера телефона).
    *   `whatsapp.verifyToken` (Токен для верификации вебхука, который мы придумаем сами).

### 2. Бэкенд (Ktor)
*   **Webhook Endpoint**: Создать `POST /webhooks/whatsapp` для получения входящих сообщений.
*   **Verification Endpoint**: Создать `GET /webhooks/whatsapp` для первоначальной проверки связи с Meta (challenge-response).
*   **WhatsAppAuthService**:
    *   Работает аналогично `TelegramAuthService`, но вместо постоянного опроса сервера (polling) он обрабатывает входящие данные от вебхука.
    *   Парсит входящий JSON, извлекает `wa_id` (номер телефона пользователя) и текст сообщения (код).
    *   Сопоставляет полученные данные с активными сессиями (`AuthSessions`).

### 3. Фронтенд (Web/Mobile)
*   **Deep Link (Ссылка)**: Использовать формат `https://wa.me/<ВашНомер>?text=login_UUID`.
*   **UI (Интерфейс)**: Добавить кнопку "Войти через WhatsApp" рядом с Telegram.

### 4. Стоимость и ограничения
*   **Стоимость**: Диалоги, инициированные пользователем (наш случай), обычно дешевле или бесплатны в рамках пакета (1000 шт/мес), но нужно проверить актуальные тарифы Meta для вашего региона (СНГ).
*   **Верификация**: Meta может потребовать официальную верификацию бизнеса (загрузка документов компании), чтобы увеличить лимиты сообщений (без верификации обычно доступно около 50-250 сообщений в сутки).
