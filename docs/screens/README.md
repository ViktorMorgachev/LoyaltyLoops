# Экранная документация

Каждый экран = пара `*_SCREEN.md` + `*_SCREEN_SPEC.md`. Правила: `../SCREEN_DOCUMENTATION_GUIDE.md`.

Модули:

- `client/` — экраны клиента (composeApp)
- `cashier/` — экраны кассира (composeApp)
- `partner/` — экраны владельца/партнёра (composeApp)
- `webadmin/` — экраны web-admin
- `shared/` — общие экраны (auth, onboarding, splash)
